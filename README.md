## 1. Resumen del proyecto
Desarrollamos un Aplicativo Web para una tienda de mascotas, a partir de Javascript Vanilla, Html y Css, donde al finalizar la compra el cliente puede validar el número de su tarjeta de crédito y cuenta con la funcionalidad de ocultar todos los dígitos de su tarjeta exceptuando los últimos cuatro números.
